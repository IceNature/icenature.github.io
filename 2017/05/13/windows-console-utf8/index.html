解决 C/C++ Windows 下控制台输出 UTF-8 字符乱码

By IceNature

发表于 2017-05-13

文章目录

在 Windows 下使用 C/C++ 向控制台输出 UTF-8 字符时总是会出现乱码。这是由于 Windows 默认使用 GBK 作为控制台字符的编码。
所以只需修改控制台的代码页并使用支持 UTF-8 编码的字体即可正常输出。
